思路

数据范围来看可以接受O(n2)做法，看题目标题的意思会有进阶题，咱就是体现一个具体问题具体分析，暴力O(n2)解题。

用pair数组存储所有的事件起止时间，新事件来时遍历数组，判断是否存在重合点，存在则返回false，不存在append新事件到数组中，返回true。

代码

class MyCalendar {
public:
    pair<int, int> events[1005];
    int tot;
    MyCalendar() {
        tot = 0;
    }
    
    bool book(int start, int end) {
        end--;
        for(int i = 0; i < tot; ++i){
            if(events[i].first > end || events[i].second < start) continue;
            return false;
        }
        events[tot++] = pair<int, int>(start, end);
        return true;
    }
};

/**
 * Your MyCalendar object will be instantiated and called as such:
 * MyCalendar* obj = new MyCalendar();
 * bool param_1 = obj->book(start,end);
 */
